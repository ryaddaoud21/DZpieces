# DZpieces
Pieces DZ est un site d'annonces en Algérie . pour les pièces détachées automobiles où  il y a 03 acteurs :
- le vendeur qui peut publier une annonce en ajoutant un produit ( modifier / supprimer produit - consulter les offres reçus - ajouter une offre ).
- le client qui peut envoyer des offres pour le vendeur concernant l'annonce ( proposer une prix , chercher des produits , contacter le vendeur ) 
- l'administrateur.
-  avec messagerie entre vendeur/client .
